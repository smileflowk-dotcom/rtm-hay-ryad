# RTM v1.0 RC

Version publique assainie du Référentiel Territorial Multifinalitaire pour le territoire pilote Hay Ryad.

Le site statique présente :

- le tableau de bord public ;
- la carte schématique des coordonnées publiées par les sources ;
- les relations validées entre objets et sources ;
- l'alerte connue concernant la source AURS bloquée ;
- l'historique récent assaini.

## Publication

Netlify publie directement le dossier `public/`. Aucune commande de build ni dépendance externe n'est nécessaire.

Les sauvegardes, journaux, preuves brutes, données métier internes, configurations locales et secrets sont exclus du dépôt public.

## Consultation locale

Servir le dossier `public/` avec un serveur HTTP statique, puis ouvrir `index.html` depuis ce serveur.
